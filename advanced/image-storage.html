图片存储和使用 ​

Keep 主题中，在需要填写图片配置项的地方，无论是主题配置文件，还是文章中，你都可以使用 本地图片路径 或 图片链接。

本地图片路径 ​

本地图片路径即存储你的 Hexo 项目根目录下 source/images/ 文件夹里面的图片路径。

使用示例：

头像图片路径：your-hexo/source/images/avatar.png

主题配置文件中使用：

yaml
avatar: /images/avatar.png
1

图片链接 ​

图片链接即你存储在线上文件服务器上的图片路径链接。

使用示例：

头像图片链接：https://xpoet.cn/images/avatar.png

主题配置文件中使用：

yaml
avatar: https://xpoet.cn/images/avatar.png
1

PicX ​

PicX 是一款基于 GitHub API 开发的图床工具，提供图片上传托管、生成图片链接和常用图片工具箱服务。

作者推荐你使用 PicX 来托管自己的图片资源。

注意 ​

为了有效提高你博客站点的访问速度，强烈建议你对所使用的图片资源进行压缩。

图片压缩工具推荐：

  • PicX 工具箱
  • TinyPNG
