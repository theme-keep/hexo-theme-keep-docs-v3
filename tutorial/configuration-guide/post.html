post ​

post 配置项用于设置文章内容页的文章字数统计、阅读时长、作者标识和图片排列位置 。

yaml
post:
  author_label:
    enable: true        # Option values: true | false
    auto: true          # Option values: true | false
    custom_label_list: ["Trainee", "Engineer", "Architect"]

  word_count:
    wordcount: false     # Option values: true | false
    min2read: false      # Option values: true | false

  datetime:
    create: true         # Option values: true | false
    update: true         # Option values: true | false
    format: "YYYY-MM-DD HH:mm:ss" # e.g. "YYYY-MM-DD HH:mm:ss"

  img_align: left        # Option values: left | center

  copyright_info: false  # Option values: true | false

author_avatar v3.7.1 已废弃 ​

📢 注意

author_avatar 配置项在 v3.7.1 已废弃。如有需要，请使用 Keep 主题的 inject 功能。

author_avatar 设置文章内容页的作者头像是否显示。

  • true 显示

  • false 隐藏

author_label ​

author_label 设置文章内容页的作者标识（称号），目前有两种方案：

  1. auto 自动生成 Lv1、Lv2、Lv3 ...... 判断依据为文章数量，Lv 级别越高，增长越慢。

    yaml
    author_label:
   enable: true
   auto: true
    1
    2
    3

  2. custom_label_list 自定义标识列表，数组项可以 一个 或 多个，匹配的依据是按文章数量。

  • 一个数组项
    不管文章数量多少，将一直显示这个数组项的标识。

    yaml
    author_label:
  enable: true
  auto: false
  custom_label_list: ["前端鼓励师"]
    1
    2
    3
    4

  • 多个数组项
    根据你的发布的文章数量，自动匹配作者标识。

    yaml
    custom_label_list: `["实习生", "工程师", "架构师"]`
    1
    yaml
    custom_label_list: `["炼气", "筑基", "结丹", "元婴", "化神"]`
    1

word_count ​

word_count 用于实现文章字数统计和计算阅读时长，使用前需在 Hexo 项目根目录下安装第三方依赖 hexo-wordcount。

sh
$ npm install hexo-wordcount
1

wordcount ​

wordcount 是否开启文章字数统计，可选值：true 或 false。

yaml
word_count:
  wordcount: true
1
2

min2read ​

min2read 是否开启文章阅读时长，可选值：true 或 false。

yaml
word_count:
  min2read: true
1
2

datetime v3.7.0 ​

datetime 用于控制文章内容页面的日期时间显示。

create v3.7.0 ​

create 用于控制显示或隐藏文章的创建时间。

update v3.7.0 ​

update 用于控制显示或隐藏文章的更新时间。

format v3.7.0 ​

format 用于设置文章的创建时间和更新时间的格式，默认为 YYYY-MM-DD HH:mm:ss。

示例：

yaml
format: "YYYY-MM-DD HH:mm"
1

img_align ​

img_align 设置文章内容页的图片排列位置，默认 left，可选：left（靠左）、center（居中）。

yaml
img_align: left  # Option values: left | center
1

  • left（靠左）

  • center（居中）

copyright_info ​

copyright_info 用于开启文章内容页的文章版权信息，可选值：true 或 false。

yaml
copyright_info: true
