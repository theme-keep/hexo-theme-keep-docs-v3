post #

post 配置项用于设置文章内容页的文章字数统计、阅读时长、作者标识和图片排列位置 。

yaml
post:
  author_label:
    enable: true  # Option values: true | false
    auto: true    # Option values: true | false
    custom_label_list: ["Trainee", "Engineer", "Architect"]

  word_count:
    enable: false       # Option values: true | false
    wordcount: false    # Word count, one article. Option values: true | false
    min2read: false     # Time to read, one article. Option values: true | false

  img_align: left  # Option values: left | center

  copyright_info: false  # Option values: true | false

author_label #

author_label 设置文章内容页的作者标识（称号），目前有两种方案：

  1. auto 自动生成 Lv1、Lv2、Lv3 ...... 判断依据为文章数量。

  2. custom_label_list 自定义标识列表，数组项可以 一个 或 多个，匹配的依据是按文章数量。

  • 一个数组项
    不管文章数量多少，将一直显示这个数组项的标识。

    yaml
    custom_label_list: ["前端鼓励师"]
    1

  • 多个数组项
    根据你的发布的文章数量，自动匹配作者标识。

    yaml
    custom_label_list: `["实习生", "工程师", "架构师"]`
    1

word_count #

word_count 用于实现文章字数统计和计算阅读时长，使用前需在 Hexo 项目根目录下安装依赖 hexo-wordcount。

sh
$ npm install hexo-wordcount
1

wordcount #

wordcount 文章字数统计，可选值：true 或 false。

min2read #

min2read 文章阅读时长，可选值：true 或 false。

img_align #

img_align 设置文章内容页的图片排列位置，默认 left，可选：left（靠左）、center（居中）。

  • left（靠左）

  • center（居中）

copyright_info #

copyright_info 用于开启文章内容页的文章版权信息，可选值：true 或 false。

yaml
copyright_info: true
