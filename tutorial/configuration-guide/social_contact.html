social_contact #

social_contact 在首屏里面配置第三方社交链接。

yaml
social_contact:
  enable: false
  links:
    github:          # GitHub
    weixin:          # WeChat
    qq:              # QQ
    weibo:           # WeiBo
    zhihu:           # ZhiHu
    twitter:         # Twitter
    facebook:        # Facebook
    email:           # Email

enable #

enable 设置是否开启显示第三方社交链接，可选值：true 或 false。

注意

social_contact 仅在首屏开启状态下才生效！

links #

在 links 下方的第三方社交平台配置项填写自己的 URL。

点击跳转外部链接 #

当我们在 links 的配置项里填写的是 URL 时，点击对应图标时会直接跳转到外部 URL。

配置示例：

yaml
github: https://github.com/XPoet
1

点击悬浮打开图片 #

当我们需要在点击图标时悬浮打开图片，例如点击微信图标，悬浮打开微信二维码时，那么需要给 URL 加上前缀 img | 。

配置示例：

yaml
weixin: img | https://x.com/images/wechat.png
1

自定义配置 #

Keep 主题内置 Font Awesome v6.2.0 免费版图标，基本已覆盖我们所需的全部图标。

如果要自定义更改图标或添加其他第三方链接，例如：添加 Instagram

  1. 在 Font Awesome 搜索 Instagram 图标。

  2. 填写图标名（如上图），添加到主题配置文件 _config.yml 的 social_contact.links 配置项里面。

yaml
social_contact:
  enable: true
  links:
    instagram: https://www.instagram.com/xxxxxxxx
